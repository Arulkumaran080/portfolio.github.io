Ticket Booking Web Application

It is dynamic and responsive web application for booking movie tickets online for different genres and languages.

It consists of Two section:

User Portal:

In this application user can do certain operation like,

  • 1. The user can able to register to the application to book tickets.
  • 2. After registering an account, the user able to update their accounts
    and able to change the password.
  • 3. Sign into the application to maintain the record of activities.
  • 4. The user can able to search for movie tickets based on the search
    keyword.
  • 5. Apply filters and sort results based on different genres.
  • 6. User can add all the selected movie ticket to a cart and customize
    the purchase at the end.
  • 7. Experience a seamless payment process.
  • 8. Receive a booking summary page once the payment is complete.

    • Admin Portal:

    In this application admin can do certain operation like,

  • 1. The admin can create account and able to login into account.
  • 2. It can able to add or remove diiferent genres to or from application
    to build a rich prroduct line.
  • 3. Admin can able to do edit movie details like name, ticket price, language,
    description and show timing to keep it aligned to the current price.
  • 4. Once admin is authenticated, the admin can be used to enable or
    disbale movie show.
  • 5. The admin can delete the movies and able to view different genres.
